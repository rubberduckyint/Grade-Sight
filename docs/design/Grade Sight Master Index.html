RULES

Five rules that settle 90% of decisions.

Print this. Tape it next to your monitor. If a screen breaks one of these, it's wrong.

1 · 18px is the floor.

Body is 18px. Density is a layout decision — never a font-size decision.

2 · Amber is precious.

Insight color only. Never chrome. Never a generic warning.

3 · Red is evidence only.

Mock student error marks. Never UI errors. Errors use ink + mono microcopy.

4 · Serif for meaning, sans for doing.

Headlines and voice are Source Serif 4. Buttons and forms are Inter. Caveat is the student's hand only.

5 · Focus rings never hide.

:focus-visible is global. Don't override outlines on any interactive element.
